\doxysubsection{Detailed Description}
Defines the \doxylink{classPage}{Page} class, which represents a Wikipedia page and its links. 

This file contains the declaration of the \doxylink{classPage}{Page} class, which models a Wikipedia page. Each \doxylink{classPage}{Page} object stores the title of the page and a list of links to other pages. The class also includes methods to fetch links from a Wikipedia page and access the page\textquotesingle{}s title and links.
