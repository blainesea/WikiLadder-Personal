\doxysubsection{Detailed Description}
Defines the \doxylink{classWikiLadder}{Wiki\+Ladder} class, which finds the shortest ladder of Wikipedia pages. 

This file contains the declaration of the \doxylink{classWikiLadder}{Wiki\+Ladder} class, which is responsible for finding the shortest path (ladder) between two Wikipedia pages using a breadth-\/first search (BFS) algorithm. The class uses the \doxylink{classPage}{Page} class to represent individual Wikipedia pages and their links.
