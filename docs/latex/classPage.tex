\doxysubsection{Detailed Description}
Represents a Wikipedia page with a title and a list of links to other pages. 

The \doxylink{classPage}{Page} class models a single Wikipedia page, including its title and the hyperlinks to other pages contained within the page. It includes methods to fetch links from the page and retrieve the title and list of links.

Fetches links from the Wikipedia page. 

Fetches the HTML content of the Wikipedia page and parses its links.

This method fetches the HTML content of the page and parses it to extract all links to other pages. The links are stored in the {\ttfamily links} member variable.

This method builds the URL for the Wikipedia page based on its title and uses c\+URL to fetch the HTML content of the page.
