\doxysubsection{Detailed Description}
Main entry point for the Wiki Ladder application. 

This file handles user input and orchestrates the creation of \doxylink{classPage}{Page} and \doxylink{classWikiLadder}{Wiki\+Ladder} objects to find and display the Wikipedia ladder between two pages.

Asks the user to input the titles of the starting and target Wikipedia pages.

The user is prompted to enter the title of the starting Wikipedia page and the target Wikipedia page.

Initializes the \doxylink{classPage}{Page} objects for the start and target Wikipedia pages.

The \doxylink{classPage}{Page} objects represent the starting and target Wikipedia pages and are initialized with the titles provided by the user.

Fetches the links from the starting Wikipedia page.

This method invokes the {\ttfamily fetch\+Links()} function on the start page to retrieve the links from the starting Wikipedia page.

Creates a \doxylink{classWikiLadder}{Wiki\+Ladder} object and attempts to find the ladder between the start and target pages.

The \doxylink{classWikiLadder}{Wiki\+Ladder} object is used to find a sequence of Wikipedia pages that links the start and target pages.

Displays the results of the Wiki Ladder search.

If a ladder is found, the sequence of pages is displayed. If no ladder is found, a message is shown.
